# Ecommerce Fullstack Design

A full-stack eCommerce application with React frontend and Express backend.

## Setup

1. Install dependencies for server: `cd server && npm install`
2. Install dependencies for client: `cd client && npm install`
3. Start MongoDB
4. Seed database: `cd server && npm run seed`
5. Start server: `cd server && npm run dev`
6. Start client: `cd client && npm start`

## Features

- Product listing and details
- Cart functionality
- User authentication
- Admin panel for product management
